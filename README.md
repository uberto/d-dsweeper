# D&D Sweeper

A simple 2D game built with Pygame and Pygbag that combines the classic Minesweeper gameplay with D&D elements.

## Development Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the game locally:
```bash
python main.py
```
